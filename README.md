## 🌟 Introduction

**HatKaraoke.vn** is a web-based karaoke application built with React and Vite. It provides a user-friendly interface for browsing songs, managing playlists, and enjoying a seamless karaoke experience directly in the browser.

## ✨ Key Features

-   **🎵 Song Library**: Browse and search for your favorite tracks.
-   **🎤  Lyrics Display**: Synchronized lyrics for singing along.
-   **📱 Responsive Design**: Optimized for both desktop and mobile devices.
-   **⚡ Fast Performance**: Powered by Vite for lightning-fast load times.

## 🛠️ Tech Stack

-   **Frontend**: [React](https://react.dev/), [Vite](https://vitejs.dev/)
-   **Language**: TypeScript
-   **Styling**: CSS (PostCSS)

## 🚀 Getting Started

### Prerequisites

-   Node.js
-   npm/yarn

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/KhanhNguyen9872/hatkaraoke.vn.git
    cd hatkaraoke.vn
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run Development Server**:
    ```bash
    npm run dev
    ```
